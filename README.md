# Leopa Log

レオパードゲッコー（ヒョウモントカゲモドキ）の  
日々の飼育状況を、確認・記録するための個人用アプリです。

飼育を「管理」したり「評価」したりするのではなく、  
今日も問題なく過ごせているかを自分で把握することを目的にしています。

---

## コンセプト

このアプリは  
「正しい飼育方法」を提示するものでも  
「理想的な記録」を求めるものでもありません。

自分とレオパの生活に、そっと寄り添う道具として設計しています。

---

## 主な機能

- 日々の飼育ログの記録
  - 給餌
  - 体重
  - 脱皮
  - 気づいたことのメモ
- カレンダー形式での履歴確認
- 過去の記録を振り返るための一覧表示

※ 通知・SNS共有・スコア化などの機能はありません。

---

## 想定している使い方

- 世話のあとに1タップで記録
- 夜に「今日は大丈夫だったか」を確認
- 不安になったときに過去ログを見返す

毎日使うことを前提にした、シンプルな導線を意識しています。

---

## このアプリがしないこと

- 飼育ノウハウの押し付け
- 正解・不正解の判断
- 他人との比較
- 飼育の点数化・評価

あくまで  
飼育している本人の感覚と判断を尊重します。

## 開発の背景

レオパの飼育を続ける中で、

- 「昨日どうだったか」を忘れる
- 問題がない日は記録が残らない
- 不安になったときに振り返れない

といった小さな不便を感じていました。

このアプリは、  
その不便を自分自身のために解消する目的で作られています。
